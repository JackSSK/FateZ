Install
=================================

FateZ can be cloned from `GitHub <https://github.com/JackSSK/FateZ/>`_ ::

  git clone https://github.com/JackSSK/FateZ.git
  cd FateZ
  pip install .


Test
=================================

After installation, a performance test can be carried out with::

  import fatez
